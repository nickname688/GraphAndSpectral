# Graph Spectra - 图谱分析工具

## 项目概述

本项目是一个基于Python开发的图谱分析软件包，主要用于解决图论中的谱极值问题。项目分为两个主要部分：`graph`（图谱基础分析）和 `spectral`（谱图理论高级计算）。它提供了一个图形化界面，允许用户交互式地构建图、设置约束条件，并计算图的最大特征值和对应的Perron-Frobenius向量。

## 使用方法

### Graph模块使用
1. 运行 `python graph/main.py`
2. 通过界面上的按钮添加/删除边和节点
3. 选择预设图结构或通过命令行输入自定义图
4. 点击"开始计算"获取特征值和Perron-Frobenius向量

### Spectral模块使用
1. 运行 `python spectral/main.py`
2. 构建禁用子图或选择预设图
3. 设置计算参数（边数/顶点数）
4. 点击"开始计算"进行谱极值问题求解
5. 在高级设置中可调整更多参数

## 核心算法

### 特征值计算
使用NumPy的`linalg.eig`函数计算邻接矩阵的特征值和特征向量，然后找出最大特征值及其对应的Perron-Frobenius向量。

### 谱极值问题求解
- **固定阶求解器**: 求解具有固定顶点数的图的最大谱半径
- **固定边数求解器**: 求解具有固定边数的图的最大谱半径
- **约束处理**: 包括禁用子图约束、平面图约束、外平面图约束等
